GitHub 发布blog pelican操作步骤

github 账户创建

参考 github pages setting

初始化

创建content分支，克隆到本地，在content分支上运行pelican-quickstart

在content下创建页面

使用markdown编辑器，例如atom，notepad++等

生成站点

$ pelican content

将output中的文件导出到master分支中

$ ghp-import -m "Generate Pelican site" --no-jekyll -b master output

推送到github
